# Nilabh Pandey — Portfolio

Personal portfolio website showcasing AI research, robotics projects, publications, and experience.

## Features

- Dark / Light mode (dark by default)
- Interactive custom cursor with glow trail
- Particle network animation background
- Scroll-triggered reveal animations
- Fully responsive design
- Sections: Hero, About, Experience, Education, Publications, Projects, Skills, Certifications, Contact

## Tech

Single-file HTML/CSS/JS — no build step, no dependencies.
